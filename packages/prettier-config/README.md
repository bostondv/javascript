# @instacart/prettier-config
[![NPM version][npm-image]][npm-url]

Instacart's shared Prettier configuration

## Installation

```bash
$ yarn add --dev @instacart/prettier-config
```

## Usage

```js
// prettier.config.js
module.exports = require('@instacart/prettier-config')
```
